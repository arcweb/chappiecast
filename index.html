HOW TO INSTALL CHAPPIECAST

  • Clone the repo
  • npm install
  • bower install
  • npm start to start the server
  • Open a few browser windows and point them at localhost:5000
  • Use the control panel at localhost:5000/control.html to arrange windows and scale the video
  • To load chappiecast on an iOS or Android device, use the cordova app (found in the app directory)
